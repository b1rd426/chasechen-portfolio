# Site Brief

## 项目定位

我要搭建一个个人技术作品集网站，用于未来实习、求职和长期记录技术成长。

我是软件工程大一学生，目前正在学习 Web 开发、数据结构、算法、数据库和 AI 应用开发。

这个网站不是普通简历页，而是一个长期更新的成长型作品集网站。

## 目标用户

- 未来的实习面试官
- HR
- 技术同学
- 我自己，用于记录学习和项目成长

## 网站目标

访问者应该能在 1 分钟内知道我是谁，在 3 分钟内看到我的项目，在 5 分钟内了解我的学习能力和动手能力。

## 页面结构

网站第一版包含以下页面：

1. 首页 Home
2. 项目页 Projects
3. 博客页 Blog
4. 实验室 Lab
5. 关于我 About

## 首页内容

首页需要包含：

- Hero 区域
- 简短个人介绍
- 精选项目
- 最新博客
- 技能栈
- 联系方式
- GitHub 链接

## 项目页内容

第一版先放 3 个占位项目：

### 1. 个人作品集网站

技术栈：Next.js / TypeScript / Tailwind CSS  
说明：用于展示我的项目、博客、实验室和学习路线。

### 2. 算法可视化实验室

技术栈：React / TypeScript  
说明：用动画展示排序、二分查找、递归、动态规划等算法过程。

### 3. AI 学习助手工具

技术栈：Next.js / API / Markdown  
说明：帮助学生整理学习笔记、生成复习计划和总结知识点。

## 博客页内容

第一版先放 3 篇占位文章：

1. 我为什么要搭建这个个人网站
2. 大一软件工程学生应该如何建立作品集
3. 第一次使用 Next.js 搭建个人网站的记录

博客分类：

- Web 开发
- 数据结构与算法
- AI 工具开发
- 项目复盘

## 实验室页面

实验室页面用于放未来的小工具、小 Demo 和技术实验。

第一版可以放这些占位卡片：

- 排序算法可视化
- GPA 计算器
- AI 简历优化器
- Markdown 笔记整理器

## 关于我页面

关于我页面需要展示：

- 我是谁
- 我的学习方向
- 我的技能栈
- 我的阶段性目标
- 联系方式

## 技术要求

请使用：

- Next.js
- TypeScript
- Tailwind CSS

可以使用：

- Framer Motion
- Lucide Icons
- Markdown 或 MDX

## 设计风格

整体风格要求：

- 简洁
- 现代
- 技术感
- 深色模式优先
- 页面留白充足
- 不要过度花哨
- 卡片式布局
- 响应式设计
- 适配手机、平板和电脑

## 第一版目标

第一版只做 MVP，不要加入复杂后端。

需要完成：

- 网站可以正常运行
- 首页完整
- 项目页完整
- 博客页完整
- 实验室页完整
- 关于我页完整
- 导航栏可用
- 移动端适配
- 可部署到 Vercel

## 暂时不要做

第一版不要做：

- 登录系统
- 数据库
- 评论系统
- 支付功能
- 后台管理
- 复杂 AI API 调用
- 用户上传文件
